# FitZone Gym Management System

A complete web-based gym management system built with PHP and MySQL.

## Features
- Member Management (Add, View, Delete)
- Trainer Management (Add, View, Delete)
- Class Management (Add, View, Delete)
- Enrollment System with capacity tracking
- Responsive design
- Full CRUD operations

## Technologies Used
- PHP 8.2
- MySQL
- HTML5/CSS3
- XAMPP

## Installation

1. Install XAMPP
2. Copy project to `htdocs/gym_project`
3. Import `database.sql` via phpMyAdmin
4. Update `includes/db_connect.php` with your database credentials
5. Access via `http://localhost/gym_project/`

## Database Configuration
- Database Name: gym_fitzone
- Default User: root
- Default Password: (empty)
- Port: 3307 (or 3306)
